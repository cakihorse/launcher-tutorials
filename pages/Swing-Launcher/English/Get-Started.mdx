# Swing Launcher Documentation.

Here you are go to find some Documentation about the code of the SwingLauncher.

## Installation
 - To create your own Launcher using the repo as a template, clone it with git into your computer.
 - Open this project with the editor you want to use. (Intelij Idea is strongly recommended but you can use other ide.) Now you are ready to code your own launcher with this repo as base.
